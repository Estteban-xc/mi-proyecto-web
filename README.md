## Guía para clonar y visualizar el proyecto de manera local

Este proyecto puede ejecutarse de forma local en cualquier equipo que tenga instalado Git y un navegador web. A continuación se describen los pasos necesarios para clonar el repositorio y visualizar el sitio.

### 1. Requisitos previos

- Tener instalado Git en el equipo.
- Contar con un navegador web actualizado (Google Chrome, Microsoft Edge, Firefox, etc.).

### 2. Clonar el repositorio

Desde GitHub, copia la URL del repositorio.  
Luego, abre una terminal o línea de comandos y ejecuta:

git clone https://github.com/TU_USUARIO/mi-proyecto-web.git

(Reemplaza TU_USUARIO por tu nombre de usuario en GitHub.)

Este comando descargará una copia completa del proyecto en tu equipo.

### 3. Acceder a la carpeta del proyecto

Una vez finalizada la descarga, ingresa al directorio del proyecto con el siguiente comando:

cd mi-proyecto-web

### 4. Visualizar el proyecto

Para visualizar el sitio web de manera local, abre el archivo index.html en tu navegador.

Puedes hacerlo de las siguientes formas:

- Haciendo doble clic sobre el archivo index.html.
- Haciendo clic derecho sobre el archivo y seleccionando "Abrir con" → Navegador.
- Abriendo la carpeta del proyecto en un editor como Visual Studio Code y utilizando una extensión como Live Server para ejecutar un servidor local.

Al abrir index.html se cargará la página principal del sitio y podrás navegar por las distintas secciones del proyecto.

## Tecnologías y conceptos aplicados en el proyecto

Durante el desarrollo de este sitio web se implementaron diferentes técnicas de HTML y CSS orientadas a la construcción de interfaces modernas, organizadas y visualmente atractivas.

### Estructuración y maquetación

Se utilizaron distintas técnicas de maquetación para organizar los elementos dentro de la página, permitiendo distribuir contenido en filas, columnas y secciones bien definidas. Esto facilita una estructura clara y una mejor experiencia de navegación.

### Centrado y alineación de contenido

Se aplicaron diferentes métodos de alineación y centrado para posicionar correctamente títulos, botones y bloques de contenido dentro de los contenedores, logrando una presentación equilibrada en la interfaz.

### Posicionamiento de elementos

Se utilizaron propiedades de posicionamiento para controlar la ubicación de ciertos componentes dentro de la página, permitiendo ubicar elementos de forma más precisa en relación con su contenedor o con otros elementos.

### Animaciones y efectos visuales

Se implementaron efectos visuales mediante transformaciones y transiciones CSS para mejorar la interacción del usuario con la página. Estos efectos permiten, por ejemplo, que tarjetas o botones reaccionen cuando el usuario pasa el cursor sobre ellos.

### Formularios de contacto

El sitio incluye un formulario que permite a los usuarios enviar información o consultas. Para ello se utilizaron distintos tipos de campos de entrada que facilitan la captura de datos como nombre, correo electrónico y mensajes.

### Integración de contenido externo

Se incorporaron elementos que permiten mostrar contenido externo dentro del sitio web, como recursos embebidos o servicios de terceros, integrados directamente en la interfaz.

### Contenido multimedia

El proyecto incluye elementos multimedia como video y audio que pueden reproducirse directamente dentro de la página, enriqueciendo la experiencia del usuario y permitiendo presentar contenido de forma más dinámica.
